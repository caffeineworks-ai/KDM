# KDM+ AI 도구 제작 및 디자인 활용 실습 결과

## 1. 개요
* **목적:** AI에 자각 도구를 연결해 디자인 문제를 해결하는 과정을 경험하여 기술 이해도 제고
* **내용:** MCP(Model Context Protocol) 서버를 제작하고, 이를 제품 디자인 및 디지털 브랜딩에 활용하는 과정을 실습함
